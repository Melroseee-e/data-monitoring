## 技术细节

### 数据采集方法
1. **交易所地址识别**: 使用 BubbleMaps API 获取 Top 500 持有者，筛选出 CEX 地址
2. **ATA 查找**: 对每个交易所地址，查找其 PUMP 代币的 Associated Token Account
3. **交易历史**: 使用 Helius RPC API 获取每个 ATA 的所有交易签名
4. **交易解析**: 解析每笔交易，识别流入/流出方向和数量
5. **数据聚合**: 按小时聚合数据，生成每日汇总

### API 使用
- **BubbleMaps API**: 获取持有者和标签信息
- **Helius RPC API**: 获取 Solana 交易历史
- **速率限制**: 150ms 延迟 + 指数退避重试

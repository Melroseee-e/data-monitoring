构建一个自动化的链上数据监控系统,实时追踪多个代币在各大交易所的资金流入流出情况,为投资决策提供数据支持。

🏗️ 技术架构

GitHub Actions 定时任务
每小时触发一次 (Cron: 0 * * * *)
Python 数据采集脚本
• 从 CoinMarketCap 获取代币合约地址
• 查询 Bitquery API 获取链上转账数据
• 识别交易所地址并计算流入流出
• 标记重要地址并生成 JSON 数据
数据存储 (JSON 文件)
• tokens.json - 代币基础信息
• exchanges.json - 交易所地址配置
• latest_data.json - 最新监控数据
• history/ - 历史数据归档
前端展示 (HTML + JavaScript)
• 交互式图表展示流入流出趋势
• 支持按交易所筛选
• 重要地址高亮显示
• 响应式设计,支持移动端
GitHub Pages 部署
免费静态网站托管,全球 CDN 加速

🔧 数据源和工具选择

Bitquery API
主数据源
优势: 支持 40+ 区块链,提供 GraphQL API,数据质量高
免费额度: 10,000 API points/月
用途: 查询链上转账、识别交易所地址、计算流入流出
CoinMarketCap API
辅助数据源
优势: 权威的代币信息,合约地址准确
免费额度: 10,000 credits/月
用途: 获取代币合约地址和基础信息
DexScreener API
补充数据源
优势: 无需 API key,完全免费
免费额度: 无限制
用途: DEX 数据补充,价格信息
Bubblemaps
可视化工具
优势: 链上地址关系可视化,交易所地址识别
用途: 辅助识别交易所地址,验证数据准确性
